# 📊 Java 2D Array Programs

This repository contains Java programs focused on **2D Arrays (Matrices)**.  
It helps in understanding how to work with rows, columns, and perform various operations on matrices using Java.

---

## 📌 Programs Included

The repository includes the following 2D array programs:

- 🔍 Search Element in Matrix (Q_Search_X.java)
- 🌀 Spiral Order Traversal of Matrix
- 📋 Basic 2D Array Input & Output (TwoDArr.java)
- 🔄 Matrix Transpose (print_transpose.java)

---

## 🛠 Technologies Used

- **Java**
- **2D Arrays (Matrices)**
- **Nested Loops**
- **Conditional Statements**
